## 【発展】配列・ポインタ配列・ダブルポインタ
以下のようなコードがあったときのダブルポインタの使い方
```c
#include<stdio.h>

void main(void){
	int b[3][3] = {
		{10,20,30},
		{40,50,60},
		{70,80,90}
	};
	int* pb[3] = { b[0],b[1],b[2] };
	int** ppb = pb;
}
```
- `int型二次元配列b`を3*3で宣言
- `int型ポインタ配列pb`を`bのポインタ`として宣言
- `int型ダブルポインタppa`を`pbのポインタ`、`bのダブルポインタ`として宣言
